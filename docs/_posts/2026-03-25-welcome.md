Welcome to **Euler AI**! Here we will share development progress, technical insights, and important announcements for the Euler AI project.

## About This Blog

This blog is built with Jekyll and hosted on GitHub Pages. It serves as a platform for:

- Publishing project progress updates
- Sharing technical implementation details
- Documenting development decisions and learnings

## How to Publish New Posts

Create a new Markdown file in the `_posts` directory with the following naming format:

```
YYYY-MM-DD-title.md
```

For example: `2026-03-25-welcome.md`

Each post should include Front Matter:

```yaml
---
layout: post
title: "Post Title"
date: 2026-03-25
author: Author Name
tags: [Tag1, Tag2]
---
```
